Detailed Description

Represents an OpenGL VertexArray.

This class provides an abstraction for managing OpenGL Vertex Array Objects (VAOs). A VAO stores the state of vertex attributes and the associated vertex buffer bindings, allowing efficient configuration and rendering of vertex data.
