Detailed Description

A class for loading, binding, and managing 2D textures in OpenGL.

This class is responsible for loading texture data from a file, creating an OpenGL texture object, and setting texture parameters such as wrapping and filtering. The texture can then be bound for rendering.

Constructs a Texture object and loads a texture from a file.

This constructor loads texture data from the specified file and creates an OpenGL texture object using the parameters defined in the TextureSpec structure.

Parameters
sourceThe file path to the texture image (e.g., PNG or JPEG file).
specA TextureSpec structure defining wrapping and filtering parameters.
Note
If the texture fails to load, the texture object will not be initialized.
