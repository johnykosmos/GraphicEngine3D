\doxysubsection{Detailed Description}
Abstract base class for cameras in a 3D graphics system. 

The \doxylink{classeng_1_1Camera}{Camera} class provides an interface for deriving different types of cameras. It includes a virtual destructor and a pure virtual method for retrieving the view-\/projection matrix.
