\doxysubsection{Detailed Description}
Represents a point light in the scene. 

A point light emits light in all directions from a single position in space. The light\textquotesingle{}s intensity diminishes with distance based on attenuation factors.

\begin{DoxyNote}{Note}
The data is aligned to meet GPU uniform buffer layout requirements. 
\end{DoxyNote}
