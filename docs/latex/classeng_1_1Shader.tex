\doxysubsection{Detailed Description}
A class for loading, compiling, and using Open\+GL shaders. 

This class is used to load and compile shaders from source files, create an Open\+GL shader program, and bind/unbind the shader program for rendering. It handles both vertex and fragment shaders.
