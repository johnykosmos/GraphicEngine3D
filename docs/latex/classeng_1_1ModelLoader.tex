\doxysubsection{Detailed Description}
A utility class for loading 3D model data from an OBJ file. 

The \doxylink{classeng_1_1ModelLoader}{Model\+Loader} class is responsible for loading and parsing OBJ files, extracting the vertex positions, normals, texture coordinates, and face indices, and storing them into a \doxylink{classeng_1_1Mesh}{Mesh} object. It uses the \doxylink{classeng_1_1ModelParser}{Model\+Parser} class to handle the parsing process and converts the parsed data into a format suitable for rendering.
