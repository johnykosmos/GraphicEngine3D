\doxysubsection{Detailed Description}
Manages the collection of lights, drawables, and the active camera in a 3D scene. 

The \doxylink{classeng_1_1Scene}{Scene} class provides methods to add, remove, and retrieve light sources and drawable entities in the scene. It also manages the active camera, which is used for rendering and scene transformations. The \doxylink{classeng_1_1Scene}{Scene} serves as a central container for all objects and components that are rendered or used during rendering.
