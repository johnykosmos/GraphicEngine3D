\doxysubsection{Detailed Description}
Represents an Open\+GL \doxylink{classeng_1_1IndexBuffer}{Index\+Buffer}. 

This class provides an abstraction for managing Open\+GL IBOs, which store indices that define the order in which vertices are rendered.
