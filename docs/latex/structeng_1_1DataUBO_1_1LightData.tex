\doxysubsection{Detailed Description}
Holds the lighting data for the scene. 

This struct contains arrays of point lights and directional lights, along with the number of lights in each category, for use in the lighting calculations.

\begin{DoxyNote}{Note}
The data is aligned to meet GPU uniform buffer layout requirements. 
\end{DoxyNote}
