\doxysubsection{Detailed Description}
A class for loading, binding, and managing 2D textures in Open\+GL. 

This class is responsible for loading texture data from a file, creating an Open\+GL texture object, and setting texture parameters such as wrapping and filtering. The texture can then be bound for rendering.
